# Sistema de Controle de Frequência / Attendance Control System

## Português

### Descrição
Sistema web para automatizar o controle de frequência de funcionários por setor, substituindo processos manuais e planilhas físicas.

### Tecnologias
- Python (Flask)
- SQLite
- HTML/CSS/JavaScript

### Estrutura
- `src/`: Código fonte da aplicação.
- `src/app.py`: Ponto de entrada do Flask.
- `src/database.db`: Banco de dados SQLite.
- `src/templates/`: Arquivos HTML.
- `src/static/`: Arquivos CSS e JavaScript.

### Instalação e Uso
1. Instale as dependências: `pip install -r requirements.txt`
2. Execute a aplicação: `python src/app.py`
3. Acesse `http://localhost:5000` no navegador.

## English

### Description
Web system to automate employee attendance control by sector, replacing manual processes and physical spreadsheets.

### Technologies
- Python (Flask)
- SQLite
- HTML/CSS/JavaScript

### Structure
- `src/`: Application source code.
- `src/app.py`: Flask entry point.
- `src/database.db`: SQLite database.
- `src/templates/`: HTML files.
- `src/static/`: CSS and JavaScript files.

### Installation and Usage
1. Install dependencies: `pip install -r requirements.txt`
2. Run the application: `python src/app.py`
3. Access `http://localhost:5000` in your browser.
